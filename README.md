Estou em formação na área de Engenharia de IA, aprendendo a construir agentes inteligentes, pipelines de machine learning e sistemas baseados em LLMs. Este repositório é tanto um diário de aprendizado quanto uma vitrine dos projetos que desenvolvo ao longo do caminho.

## Jornada de aprendizado

### Fundamentos
- [x] Pensamento computacional, lógica e fundamentos da computação
- [ ] Python com IA Aplicada

### Nível 1 — Construção de Agentes Inteligentes com RAG
- [ ] LangChain e Python: criando ferramentas com a OpenAI
- [ ] Arquiteturas RAG com LLMs: embeddings, busca semântica e agentes
- [ ] LangChain: Técnicas Avançadas de RAG
- [ ] LangGraph: Orquestrando agentes e multiagentes
- [ ] Protocolos e arquitetura para construção de agentes (MCP, A2A, AG-UI, BFA)
- [ ] Clusterização: lidando com dados sem rótulo
- [ ] NLP: análise de sentimentos com processamento de linguagem natural

### Nível 2 — Machine Learning, Deep Learning e Fine Tuning
- [ ] Redes Neurais Deep Learning com PyTorch
- [ ] Treinando uma Rede Neural com PyTorch
- [ ] Redes Neurais Recorrentes (RNN) com PyTorch
- [ ] Redes Neurais Convolucionais (CNN) com PyTorch
- [ ] Hugging Face: explorando e aplicando modelos de IA
- [ ] Modelos de Difusão: fundamentos e aplicações avançadas
- [ ] Transformers: fundamentos e prática com PyTorch

### Nível 3 — MLOps e AIOps: operando sistemas de IA em produção
- [ ] Inteligência Artificial: preparação para o mercado
- [ ] Empreendendo com IA: da ideação ao modelo de negócio
- [ ] Observabilidade para LLMs com LangFuse
- [ ] Pipelines de IA em Cloud: Azure, AWS e GCP
- [ ] MLFlow: gerenciamento de experimentos e IA generativa
- [ ] Integração contínua para LLMs com GitHub Actions e DeepEval
- [ ] Data Science: Regressão Linear
- [ ] Hugging Face: transferência de aprendizado em NLP
